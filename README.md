# Biology_lab_project

A quick biology lab project for the end of the semester.

**The Red Queen's Race** — an interactive website about the Red Queen Hypothesis, built for the BIOL 1003 Lab final presentation.

## Development

```bash
npm install
npm run dev      # start dev server
npm run build    # production build
```

Built with React + Vite, Tailwind CSS, and Framer Motion.
